## Overview

**StratumFi unlocks Bitcoin liquidity without forcing holders to sell or risk liquidation.**

Users deposit BTC as collateral, which is automatically paired with protocol-owned MUSD and deployed into yield-generating liquidity pools on Mezo's Tigris DEX. Against this productive collateral, users can instantly mint **bMUSD**â€”a synthetic stablecoin representing their debt. 

The yield generated by their Bitcoin automatically pays down this debt over time, creating a truly **self-repaying loan**.

2. **Traditional DeFi loans** â†’ Pay high interest (5-20% APR) + constant liquidation risk

**Your most valuable asset is effectively trapped.**

## The Solution

**StratumFi lets you take a loan that pays itself back.**

1. **Deposit BTC** â†’ Protocol puts it to work earning yield in Mezo's native MUSD/BTC pool
2. **Borrow bMUSD** â†’ Instantly mint stablecoins against your *future* yield
3. **Auto-Repayment** â†’ Yield automatically pays down your loan over time
4. **Get Bitcoin Back** â†’ After debt is repaid, withdraw your full principal + keep earning

**You get liquidity today. Your yield repays the loan. You never pay interest.**

| Contract | Purpose | Key Functions |
|----------|---------|---------------|
| **VaultController** | User entry point | `deposit()`, `withdraw()` |
| **StrategyBTC** | Primary yield source | Manages MUSD/BTC LP position on Tigris |

- `IPyth` - Pyth Network oracle for BTC pricing
- `ITigrisRouter` - Tigris DEX swaps and liquidity

## Getting Started

### Prerequisites

- **Node.js** 18+ and npm
- **Python** 3.9+ (for keeper bot)
- **MetaMask** or compatible Web3 wallet
- **Mezo Testnet** BTC for gas (get from faucet)

### 1. Smart Contracts

```bash
cd stratum-contracts

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add your private key and RPC URL

# Compile contracts
npx hardhat compile

# Deploy to Mezo Testnet
npx hardhat ignition deploy ignition/modules/StratumDeployFinal.ts --network mezoTestnet

# Run interaction scripts
npx hardhat run scripts/interact/01-deposit-btc.ts --network mezoTestnet
npx hardhat run scripts/interact/02-borrow-bmusd.ts --network mezoTestnet
```

See [`stratum-contracts/QUICKSTART.md`](stratum-contracts/QUICKSTART.md) for detailed instructions.

### 2. Frontend

```bash
cd stratum-frontend

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add contract addresses and RPC URL

# Run development server
npm run dev

# Build for production
npm run build
npm start
```

Access at [http://localhost:3000](http://localhost:3000)

### 3. Keeper Bot

```bash
cd offchain-keeper-bot

# Setup
make install
make config
# Edit .env with keeper private key
```

## Why Mezo?

StratumFi is **only possible on Mezo** for three critical reasons:

### 1. Native BTC Yield
Mezo's Tigris DEX provides deep MUSD/BTC liquidity pools where Bitcoin can earn trading fees. This is the yield engine that powers self-repaying loans.

### 2. MUSD Ecosystem
MUSD is Mezo's native stablecoin and the backbone of our protocol:
- **Protocol-Owned Liquidity:** StratumFi vaults are pre-funded with MUSD to instantly pair with user BTC deposits
- **Stable Anchor:** Our synthetic debt token (bMUSD) is pegged 1:1 to MUSD for reliability
- **Gateway Asset:** MUSD connects StratumFi to the wider Mezo DeFi landscape (Turbo Loop strategies)

### 3. Robust Oracle Infrastructure
Mezo's built-in support for **Pyth Network** oracles gave us secure, real-time BTC price feeds from day one. This is critical for safe collateralization.

**Mezo is the Bitcoin L2 designed for DeFi. StratumFi is DeFi designed for Bitcoin.**
